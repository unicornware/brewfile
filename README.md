## Getting Started

[Homebrew Bundle][1] configuration file. Managed with [homebrew-file][2].

## Usage

### Git Clone

```zsh
git clone https://github.com/unicornware/brewfile ~/.config/brewfile
brew bundle
```

### [Git Managed Brewfile][3]

```zsh
brew file set_repo -r unicornware/brewfile
brew bundle
```
